# HSM技术书 - 从思想实验到安全基石

一本从岩画密码学到HSM集成实战的完整开源技术书。

## 项目简介

本项目是一个完整的HSM（Hardware Security Module，硬件安全模块）技术教程，采用循序渐进的教学方式，带领读者从零开始掌握密码硬件安全技术。全书分为六个章节：

- **第一章**：从岩画开始，探索人类通信的安全演进历程
- **第二章**：建立HSM世界观，认识这个"数字保险箱"
- **第三章**：深度解析PKCS#11标准，学习HSM的"普通话"
- **第四章**：走进SoftHSM2源码，理解开源实现的设计精髓
- **第五章**：亲手实现hsm-lite，从理论到实践的飞跃
- **第六章**：真实项目集成经验，掌握SDK集成的实战技能

无论你是汽车安全工程师、嵌入式开发者，还是对密码学感兴趣的技术爱好者，这本书都将为你打开一扇通往硬件安全世界的大门。

## 为什么学习HSM？

在当今数字化世界中，硬件安全模块无处不在：

- **汽车安全**：车规级HSM保护密钥、执行安全启动、实现UDS诊断
- **金融支付**：银行卡交易签名、密钥安全存储
- **云计算**：云服务密钥管理、数据加密保护
- **物联网**：设备身份认证、安全通信
- **区块链**：私钥安全存储、交易签名

PKCS#11作为HSM的标准接口，是密码硬件世界的"普通话"，掌握它意味着打开了所有HSM产品的大门。

## 书籍特色

### 1. 知识闭环完整

从第一章的"岩画上的第一条信息"到第六章的"真实SDK集成"，形成完整的知识闭环：

```
理论（为什么需要安全）
    ↓
概念（HSM是什么）
    ↓
规范（PKCS#11标准）
    ↓
源码（SoftHSM2实现）
    ↓
实践（hsm-lite实现）
    ↓
实战（真实项目集成）
```

### 2. 深度解析规范

第三章基于PKCS#11 v3.1规范，系统性解析：
- Slot/Token/Session/Object对象模型
- 几百个函数的完整生命周期
- AES/RSA/ECC等机制详解
- 安全属性的组合实践

### 3. 源码级理解

第四章基于SoftHSM2开源项目，分析：
- SoftHSM.cpp如何实现PKCS#11函数
- Slot/Session/Handle管理模块
- CryptoFactory双后端架构
- ObjectStore存储设计

### 4. 手把手实践

第五章提供约600行的轻量级实现（hsm-lite）：
- 完整的PKCS#11核心流程
- 真正可编译运行的代码
- 多架构支持（x86/ARM64/ARM32）
- 清晰的代码注释

### 5. 真实集成案例

第六章基于真实项目经验，分享：
- SDK三种集成方案对比
- 三层驱动架构设计
- I2C/SPI通信踩坑与修复
- AUTOSAR UDS服务实现
- TCP命令接口设计

## 快速开始

### 在线阅读

直接浏览GitHub上的Markdown文件，按章节顺序阅读即可。

### 本地阅读

```bash
git clone https://github.com/你的用户名/hsm-book.git
cd hsm-book
```

推荐使用支持Markdown渲染的工具：
- VS Code + Markdown Preview Enhanced插件
- Typora
- Obsidian

### 运行示例代码

第五章配套的hsm-lite实现可以实际运行：

```bash
cd hsm-lite
make

# 运行测试
./hsm_test

# ARM64编译（如需交叉编译）
make arm64
```

详细说明请参考 `hsm-lite/README.md`。

## 学习建议

### 对于初学者

推荐学习路径：
1. 先读第一章，建立对安全通信演进的直观理解
2. 学习第二章，认识HSM的基本概念
3. 学习第三章基础部分（3.1-3.7），掌握PKCS#11核心概念
4. 运行第五章的hsm-lite，动手实践
5. 回到第三章深入学习函数和机制

### 对于有一定基础者

推荐学习路径：
1. 直接阅读第三章，全面理解PKCS#11规范
2. 深入第四章SoftHSM2源码分析
3. 运行和调试hsm-lite
4. 学习第六章集成实战经验

### 对于工程实践者

推荐学习路径：
1. 快速浏览第一、二章建立概念
2. 重点学习第六章SDK集成实战
3. 参考第四章源码分析理解实现
4. 根据实际需求扩展hsm-lite功能
